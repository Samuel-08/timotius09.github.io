 ##  Timotius GitHub Pages

[Repo Timotius GitHub Pages](https://github.com/Samuel-08/timotius09.github.io)

### Apa Itu
Ini adalah repositori buat situs web pribadi yang di-hosting di GitHub Pages. Situs ini nampilin portofolio, blog, dan info kontak.

### Fitur-fitur
- Halaman portofolio
- Blog dengan artikel dan tulisan
- Info kontak

### Cara Install
Buat jalanin situs secara lokal:
1. Clone repositori:
    ```sh
    git clone https://github.com/Samuel-08/timotius09.github.io.git
    ```
2. Masuk ke direktori proyek:
    ```sh
    cd timotius09.github.io
    ```
3. Buka file `index.html` di browser kamu.

### Cara Pakai
Gunakan situs ini buat nampilin portofolio dan artikel blog. Kamu bisa edit konten dengan ngupdate file HTML, CSS, dan JavaScript yang ada.

### Kebutuhan
- HTML
- CSS
- JavaScript

1. **Fork Repository**: Pertama, fork repositori proyek ke akun GitHub kamu.
2. **Clone Secara Lokal**: Clone repositori yang sudah di-fork ke mesin lokal kamu pakai Git.
   ```sh
   git clone https://github.com/Samuel-08/RevoU
   ```
3. **Buat Branch Baru**: Selalu kerjain di branch baru dengan nama yang deskriptif.
   ```sh
   git checkout -b new-feature-x
   ```
4. **Lakukan Perubahan**: Kembangkan dan uji perubahan kamu secara lokal.
5. **Commit Perubahan**: Commit dengan pesan yang jelas menjelaskan update kamu.
   ```sh
   git commit -m 'Implementasi fitur baru x.'
   ```
6. **Push ke GitHub**: Push perubahan ke repositori yang sudah di-fork.
   ```sh
   git push origin new-feature-x
   ```
7. **Kirim Pull Request**: Buat PR ke repositori proyek yang asli. Jelasin perubahan dan motivasi di baliknya.

Setelah PR kamu direview dan disetujui, itu akan digabung ke branch utama.
